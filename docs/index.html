Installation

Install Valeria globally via npm:

$ npm install -g valeria-rss-reader

Requires Node.js 20 or later.

Agent Setup

Copy this prompt to have Claude set up Valeria for you:

Install and configure Valeria RSS reader for me:
1. Run `npm install -g valeria-rss-reader`
2. Create ~/.valeria/config.json with my RSS feeds (ask me which feeds I want)
3. Run `valeria hooks` to install Claude Code notification hooks
4. Start the server with `valeria` in the background
5. Tell me the URL to open in my browser

Quick Start

  1. Run valeria setup to configure your feed providers
  2. Run valeria hooks to install Claude Code hooks
  3. Run valeria to start the server
  4. Open http://localhost:3847 in your browser

That's it! Read articles while Claude works, and you'll be notified when it's ready.

Configuration

Config is stored at ~/.valeria/config.json

{
  "port": 3847,
  "providers": {
    "readwise": {
      "enabled": true,
      "token": "your-readwise-token"
    },
    "rss": {
      "feeds": [
        { "name": "Hacker News", "url": "https://news.ycombinator.com/rss" },
        { "name": "lobste.rs", "url": "https://lobste.rs/rss" }
      ]
    }
  }
}

Providers

Readwise Reader

Connect to your Readwise Reader library. Get your API token from readwise.io/access_token.

RSS Feeds

Add any RSS or Atom feed URL. Valeria parses both formats automatically.

FAQ

How do I get notified when Claude is ready?

Run valeria hooks to install hooks into Claude Code. When Claude finishes a task or needs your attention, Valeria will show a notification with a gentle chime. The notification also shows which project triggered it (toggle with üìÅ).

Can I disable the sound?

Yes! Click the üîî button in the header to toggle sounds on/off. Your preference is saved.

Can I hide the project name in notifications?

Yes! Click the üìÅ button to toggle showing which project triggered the notification.

Why aren't full articles showing?

Some feeds only provide summaries. Click "Continue reading on original site" to see the full article. Readwise's API returns summaries for the list view.

Can I use this without Readwise?

Absolutely. You can use any RSS feed. Just add them to your config or run valeria setup.

Does this work with other AI coding tools?

Currently Valeria is designed for Claude Code's hook system. Support for other tools may be added in the future.
